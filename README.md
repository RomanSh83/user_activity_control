# User Activity Control Telegram Bot

Приложение User Activity Control Bot представляет из себя шуточный Telegram Bot,
который позволяет публиковать различные реакции на активность разных групп пользователей

Бот позволяет публиковать сообщения при проявлении активности пользователя, указанного в настройках,
а также при отсутствии его активности в течение определенного промежутка времени

---

## Подготовка к запуску проекта

### Установка необходимых зависимостей
1. Перейти в каталог, в котором будет развернут проект и выполнить команду:
```bash
git clone https://github.com/RomanSh83/user_activity_control.git
```
2. Установите менеджер проектов uv (пример для Linux):
https://docs.astral.sh/uv/getting-started/installation/
3. (Опционально) Проверьте наличие утилиты make и при необходимости установите ее


### Конфигурации

В данной версии файлы user_types.yaml, users_settings.yaml необходимо заполнить вручную

1. Перейдите в каталог /config
2. Создайте файл `base_settings.yaml` из файла `base_settings_example.yaml`
3. Создайте файл `user_types.yaml` из файла `user_types_example.yaml`
4. Создайте файл `users_settings.yaml` из файла `users_settings_example.yaml`
5. Заполните содержимое файлов (п.2-4) следуя инструкции и примерам, размещенным в файле
6. Добавьте бота в необходимые чаты в приложении Telegram и предоставьте ему права администратора
для возможности просмотра сообщений

### Используя make

Для возможности запуска инструментов разработчика - выполнить команду:

```bash
make install_dev
```

Будет инициализировано виртуальное окружение,
установлены пакеты (в том числе инструменты lint и pytest) и подключен pre-commit

### Вручную

1. Инициализация проекта:
    ```bash
    uv sync
    ```
2. Подключение pre-commit:
    ```bash
    uv sync --all-groups
    uv run pre-commit install
    ```
